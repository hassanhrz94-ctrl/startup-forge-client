# 🚀 StartupForge - Startup Team Builder Platform

StartupForge is a modern full-stack web application that connects startup founders with talented collaborators. Founders can create startup profiles, post team opportunities, and recruit developers, designers, marketers, and other professionals. Collaborators can browse startups, apply for opportunities, and track their application status. The platform also includes a powerful admin dashboard for managing users, startups, and platform activities.

# 🚀 Installation

Clone the repositories

```bash
git clone <client_repository_url>
git clone <server_repository_url>
```

Install dependencies

```bash
npm install
```

Create a `.env.local` (frontend) and `.env` (backend) file and add the required environment variables.

Run the development server

```bash
npm run dev
```
